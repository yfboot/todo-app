# Todo App 部署检查清单

## 部署前检查

### 代码质量
- [ ] 所有测试通过
- [ ] 代码已通过 lint 检查
- [ ] 没有控制台错误和警告
- [ ] 代码已格式化

### 配置检查
- [ ] package.json 中的脚本配置正确
- [ ] 环境变量已正确设置
- [ ] 构建输出目录配置正确
- [ ] 路由配置正确（如使用 SPA）

### GitHub Actions 配置
- [ ] 工作流文件语法正确
- [ ] 触发条件配置正确
- [ ] 权限设置正确
- [ ] Secrets 已配置（如需要）

### 部署平台配置
- [ ] GitHub Pages 设置已启用
- [ ] 或 Vercel/Netlify 项目已创建
- [ ] 域名配置正确（如使用自定义域名）

## 部署后验证

- [ ] 网站可以正常访问
- [ ] 所有功能正常工作
- [ ] 响应式设计正常
- [ ] 性能指标正常
- [ ] 没有控制台错误

## 常见问题

1. **404 错误**：检查路由配置和 base 路径
2. **资源加载失败**：检查资源路径和 publicPath
3. **环境变量未生效**：检查构建时的环境变量注入
4. **部署缓慢**：优化构建缓存配置
